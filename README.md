# SQL Tool - 텍스트 포맷 변환기

텍스트 포맷 변환 및 OCR 기능을 제공하는 웹 애플리케이션입니다.

## 기능

- 텍스트 포맷 변환
- OCR (이미지에서 텍스트 추출)
- 다양한 텍스트 처리 기능

## 배포

이 프로젝트는 GitHub Pages를 통해 자동으로 배포됩니다.

- **배포 URL**: https://habitree.github.io/sql_tool/

## 로컬 실행

1. 저장소 클론
```bash
git clone https://github.com/habitree/sql_tool.git
cd sql_tool
```

2. 브라우저에서 `index.html` 또는 `text_formatter.html` 파일 열기

## GitHub Pages 활성화 방법

1. GitHub 저장소의 **Settings** 탭으로 이동
2. 왼쪽 메뉴에서 **Pages** 선택
3. **Source** 섹션에서:
   - **Deploy from a branch** 선택
   - **Branch**: `gh-pages` 또는 **GitHub Actions** 선택
4. **Save** 클릭

GitHub Actions 워크플로우가 자동으로 배포를 처리합니다.
